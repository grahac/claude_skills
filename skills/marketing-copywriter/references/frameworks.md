# Writing Frameworks

## AIDA Framework
```
Attention: Hook with bold statement or question
Interest: Develop with relevant details
Desire: Build want through benefits and proof
Action: Clear next step
```

## PAS Framework
```
Problem: Identify and agitate the pain
Agitation: Make them feel it
Solution: Present your answer
```

## Before/After/Bridge
```
Before: Current painful state
After: Desired future state
Bridge: Your product/service
```

## Features to Benefits
```
Feature: What it is
Advantage: What it does
Benefit: Why that matters to them

Example:
Feature: 256-bit encryption
Advantage: Military-grade security
Benefit: Your data stays yours, always
```
