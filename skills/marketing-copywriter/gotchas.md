# Marketing Copywriter Gotchas

Common failure patterns. Update this file every time something goes wrong.

## 1. Writing AI slop despite the explicit anti-pattern list
**Problem:** Output still contains "elevate", "seamless", "unlock your potential" even though the skill bans these words.
**Fix:** After drafting, do a final pass specifically checking against `references/avoid-ai-patterns.md`. Search-and-destroy every flagged word.

## 2. Writing for everyone instead of someone
**Problem:** Copy that tries to appeal to "startups and enterprises alike" — speaks to nobody.
**Fix:** Always ask who the specific audience is. If the user says "everyone," push back and help them narrow it.

## 3. Leading with features instead of benefits
**Problem:** "256-bit encryption with AES-GCM" instead of "Your data stays yours."
**Fix:** Use the Feature → Advantage → Benefit framework from `references/frameworks.md`. Always lead with the benefit.

## 4. Burying the CTA or making it vague
**Problem:** "Learn more" at the bottom of a wall of text.
**Fix:** Every piece needs one clear, specific CTA. "Watch the 2-minute demo" beats "Learn more." Place it prominently.

## 5. Generating copy without gathering context first
**Problem:** Writing landing page copy before knowing who the audience is, what the product does, or what action is desired.
**Fix:** Always fill out the brief template from `assets/copy-brief-template.md` before writing a single word.
