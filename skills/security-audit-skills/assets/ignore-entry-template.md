# Ignore List Entry Format

Add to the appropriate section in `~/.claude/skill-audit-ignore.md`:

```markdown
### item-name
- **Reason:** [user's reason for accepting this item]
- **Reviewed:** [today's date]
- **Reviewer:** [username]
```

## Sections in ignore file
- `## Ignored Skills` — skills that have been reviewed and accepted
- `## Ignored Plugins` — third-party plugins that have been reviewed
- `## Ignored Findings` — specific findings that have been acknowledged
